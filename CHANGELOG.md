## [0.1.0] - 2026-02-22

### 新增
- 完整的游戏规则引擎
- 2-4人游戏支持
- 简单AI对手
- 基础UI界面
- 响应式设计（桌面+移动端）
- 简化卡牌数据（60张）
- GitHub Actions自动部署

### 游戏功能
- 拿取宝石（3种不同/2种相同）
- 保留卡牌（获得黄金）
- 购买卡牌（使用宝石和加成）
- 贵族访问机制
- 游戏结束判定
- 获胜者计算

### AI系统
- 简单AI：随机策略 + 优先购买有点数卡牌
- 决策时间 <500ms

### UI组件
- 游戏设置页（玩家人数、AI难度）
- 游戏主面板（卡牌区、宝石池、贵族区）
- 玩家信息面板（点数、宝石、卡牌）
- 游戏结束页（获胜者、得分、统计、重新开始）
- 通用组件（Button、Card、GemToken）

### 开发工具
- TypeScript严格模式
- ESLint代码检查
- Prettier代码格式化
- GitHub Actions CI/CD（测试+部署）

### 文档
- 完整的游戏规则文档（中文）
- 开发和使用指南
- 故障排查文档
- MIT开源协议

### 注意事项
- 当前版本为观战模式，AI自动执行所有回合
- 人类玩家交互功能计划在v0.2.0实现
- 中等和困难AI使用相同的简单AI策略

### 技术栈
- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- Zustand（状态管理）
- Framer Motion（动画）
- Vitest + React Testing Library（测试）

## 版本说明

### 版本号规则

- **主版本号 (X.0.0)**: 不兼容的API修改
- **次版本号 (0.X.0)**: 向下兼容的功能新增
- **修订号 (0.0.X)**: 向下兼容的问题修正

### 变更类型

- **新增 (Added)**: 新功能
- **变更 (Changed)**: 已有功能的变更
- **弃用 (Deprecated)**: 即将移除的功能
- **移除 (Removed)**: 已移除的功能
- **修复 (Fixed)**: Bug修复
- **安全 (Security)**: 安全漏洞修复
